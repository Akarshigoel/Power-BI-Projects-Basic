This repository contains my complete end-to-end Power BI project built using SQL Server, Power BI Desktop, Power Query, and Power BI Service.
It includes data loading, modeling, visual creation, DAX measures, report publishing, scheduled refresh, and security configurations (RLS).
Features Implemented: 
1. Power BI Desktop
- Loading data from SQL Server
- Data profiling & cleaning in Power Query
- Creating relationships & building the data model
- Creating calculated columns and DAX measures
- Adding slicers, text boxes & cards
- Creating new card visuals
- Multi-row cards & ribbon charts
- Bar charts, line charts, donut charts & matrix visuals
- Drill-through filters
- Sentiment analysis in Power Query

2. Sentiment Analysis Module
- Extracted textual data
- Cleaned & transformed data using Power Query
- Applied sentiment scoring
- Added sentiment visuals (bar charts, KPIs, conditional color formatting)

3. Publishing to Power BI Service
- Published report from Power BI Desktop
- Created workspace in Power BI Service
- Configured a gateway connection
- Implemented scheduled data refresh
- Tested and validated refresh results

4. Security & Access Control (RLS)
- Created roles in Power BI Desktop
- Implemented Row-Level Security (RLS)
- Tested RLS in Power BI Desktop and Power BI Service

5. Technologies Used
- Power BI Desktop
- Power Query
- SQL Server / SSMS
- DAX
- Power BI Service
- On-premises Data Gateway
